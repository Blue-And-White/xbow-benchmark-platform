下方示例中 $KEY 替换为你的 api-key。

方法路径鉴权说明
POST/api/auth/register注册码注册账号 {registration_code, username, password}
POST/api/auth/login用户名密码登录(返回会话cookie)
GET/POST/api/sheets登录列/建解题表(返回 api_key)
DELETE/api/sheets/{id}登录删表(停跑着的容器+级联删记录)
GET/api/challengesX-API-Key列104题+本表状态
POST/api/challenges/{benchmark}/startX-API-Key起靶机(注入随机flag,返回 url)
POST/api/challenges/{benchmark}/submitX-API-Key提交 {flag} -> {correct}
POST/api/challenges/{benchmark}/stopX-API-Key手动关闭(清flag+计时)
GET/api/leaderboard公开排行榜

curl 示例

# 启动一题
curl -X POST http://localhost:8000/api/challenges/XBEN-001-24/start \
  -H "X-API-Key: $KEY"

# 提交 flag
curl -X POST http://localhost:8000/api/challenges/XBEN-001-24/submit \
  -H "X-API-Key: $KEY" -H "Content-Type: application/json" \
  -d '{"flag":"FLAG{...}"}'

# 访问靶机环境(反代地址, attempt_id 来自启动返回)
curl http://localhost:8000/c/<attempt_id>/
